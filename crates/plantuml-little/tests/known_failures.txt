# Reference-test fixtures that currently diverge from the Java PlantUML

# landed (PR #6: scripts/wasm-dot-wrapper.sh + graphviz-anywhere-web@0.1.7
# + DejaVu Sans via fontconfig), Graphviz-version drift and font-metric
# drift have been eliminated — so every fixture listed below now reflects
# a real plantuml-little vs Java PlantUML implementation gap worth fixing.

# This file is consumed by scripts/check_reference_failures.sh which
# both .github/workflows/ci.yml and .github/workflows/release.yml invoke.
# Both require the failing set to match this list EXACTLY — no silent
# FAILED results, and no silently-healed regressions either.
#
# If you:
#   - Fix a fixture: remove its line and commit.
#   - Introduce a regression: FIRST fix the regression, DO NOT paper it
#     over by adding it here.
#
# Keep the list sorted. One test name per line. '#' prefix for comments.
